# PicBedOfGithub
A simple tool to make Github to a picturebed
## 使用之前
**必须先配置init.txt文件**

第一行是**本地**的git仓库地址
第二行是**在线**的git仓库地址的前缀

举个栗子：
>/home/andy/myGIT/PictureBed

>https://github.com/Dawn-K/PictureBed/raw/master/

>(请不要吐槽一个有斜杠一个没有斜杠
而且**第二行行末不要加空格！**

- 文件路径中**不要有空格和括号**
- 本脚本目前只支持Linux系统，后续更新支持Windows系统
